## 🚀 How to Run (Super Easy)

1. Download or clone this folder
2. Put your favorite romantic song as **`music.mp3`** in the folder
3. Open **`index.html`** in any browser (Chrome recommended)
4. Tap **"Tap to Open 💕"** and watch the magic!

**Works perfectly on mobile** — just send the link to Ankita!
